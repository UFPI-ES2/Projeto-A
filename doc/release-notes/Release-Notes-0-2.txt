PROCEDIMENTO PARA EXECUTAR A APLICAÇÃO USANDO A IDE
------------------------------------------------------------------------------------------------------------------------------------------
1. Selecione o Projeto
2. Clique com o botão direito sobre a Classe selecionada e escolha a opção "Run As -> PyDEv: Django"
3. A aplicação deverá exibir o 'status' do servidor
4. Abra um navegador web
5. digite o seguinte URL 'http://127.0.0.1:8000/' e pressione enter.
